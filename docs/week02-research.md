=====================REST API=====================  
--------------------------Kiến trúc REST API--------------------------

- REST API(Representational State Transfer API) là phương thức giao tiếp giữa client và server thông qua HTTP
- Các method thường dùng:
  GET: Lấy dữ liệu
  POST: Tạo dữ liệu
  PUT: Cập nhật toàn bộ dữ liệu
  PATCH: Cập nhật 1 phần dữ liệu
  DELETE: Xóa dữ liệu

- Ví dụ:
  GET /products
  POST /products
  PUT /products/{id}
  DELETE /products/{id}
